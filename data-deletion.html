Exclusão de Dados do Usuário

Última atualização: 22 de maio de 2026

Este aplicativo (axis-agents) coleta e armazena dados mínimos via API do Meta para operação interna da Axis Soluções Digitais. Caso queira solicitar a exclusão dos seus dados, siga as instruções abaixo.

Como solicitar a exclusão

Por email:
Envie uma mensagem para brunordumont@gmail.com com o assunto "Exclusão de dados — axis-agents" informando seu nome de usuário do Instagram ou Facebook.

Processaremos sua solicitação em até 7 dias úteis.

O que será excluído

  • Tokens de acesso associados à sua conta
  • Dados de métricas e insights armazenados localmente
  • Qualquer dado de perfil coletado via API

Confirmação

Após a exclusão, você receberá um email de confirmação no endereço utilizado para a solicitação.
