Política de Privacidade

Última atualização: 22 de maio de 2026

Esta Política de Privacidade descreve como o aplicativo axis-agents, desenvolvido e operado pela Axis Soluções Digitais, coleta, usa e protege as informações obtidas através da API do Meta (Facebook e Instagram).

1. Informações que coletamos

O aplicativo acessa as seguintes informações via API do Meta, exclusivamente para uso interno e operacional:

  • Dados de campanhas de anúncios (Meta Ads)
  • Métricas de desempenho de conteúdo (Instagram Insights)
  • Informações básicas do perfil do Instagram (nome de usuário, contagem de seguidores)

2. Como usamos as informações

As informações coletadas são utilizadas exclusivamente para:

  • Monitoramento e otimização de campanhas publicitárias
  • Análise de desempenho de conteúdo orgânico
  • Publicação automatizada de conteúdo aprovado pelo titular da conta

3. Compartilhamento de dados

Não compartilhamos, vendemos ou transferimos suas informações para terceiros. Os dados são utilizados exclusivamente de forma interna pela Axis Soluções Digitais para operação do aplicativo.

4. Armazenamento e segurança

Os dados são armazenados localmente nos sistemas da Axis Soluções Digitais, protegidos por controles de acesso. Tokens de acesso são armazenados em ambiente seguro e nunca expostos publicamente.

5. Retenção de dados

Os dados são retidos apenas pelo tempo necessário para operação do aplicativo. Dados de insights são sobrescritos a cada atualização semanal.

6. Seus direitos

Você pode solicitar a exclusão de todos os seus dados a qualquer momento através do nosso processo de exclusão de dados ou pelo email de contato abaixo.
